# vending-machine-in-JavaScript

Vending Machine written in javaScripts with functions to :
* printing inventory
* refilling inventory
* re-supplying change
* dispensing inventory based on payment
* returning change as coins (eg. $0.35 is 1 quarter and 1 dime)
